# Excel Learning Projects 

Zbiór ćwiczeń z Microsoft Excel obejmujący formuły i tabele przestawne.

## Formuły podstawowe
- **SUM, AVERAGE, MAX, MIN, COUNT, COUNTA** – podstawowe działania na danych
- **MAX(daty)** – znajdowanie najpóźniejszej daty
- **ISNUMBER** – sprawdzanie typu danych

## Formuły warunkowe
- **IF, AND, OR, IFS** – logika warunkowa
- **SUMIFS, AVERAGEIFS, MAXIFS, MINIFS** – obliczenia z wieloma warunkami
- **IFNA** – obsługa błędów #N/A

## Wyszukiwanie danych
- **VLOOKUP** – wyszukiwanie pionowe
- **XLOOKUP** – nowoczesne poziome

## Tabele przestawne
- Tworzenie i konfiguracja tabel przestawnych
- Grupowanie i filtrowanie danych
- Podsumowania i agregacje

## Technologie
- Microsoft Excel
